## Estonian - English dictionary

### Full-stack web application using Spring Boot(REST API) & Angular.

#### Used Technologies and Tools:

## Front-end

* TypeScript

* Angular 11

* Bootstrap 4

## Back-end

* Java 11

* Spring-Boot 2.4.5 (Spring REST/ Spring Data Jpa)

* Hibernate

* H2 Database
